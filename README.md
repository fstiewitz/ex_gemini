# Gemini

Gemini server implementation in Elixir.

## Installation

The package can be installed
by adding `gemini` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:gemini, git: "https://github.com/fstiewitz/ex_gemini.git", tag: "0.1"}
  ]
end
```
